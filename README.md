Данный проект является дипломной работой по изучению курса "Разработка игр на Unity" онлайн-школы XYZ https://www.school-xyz.com/unity

Ассеты и звуки предоставлены школой\
Музыка сгенерирована через Suno\
Изображение корабля, иконка перка оглушения и иконка перка меча сгенерированы через PixelLab

Управление:
- WASD - движение 
- Левая кнопка мыши - нанести удар мечом
- E - взаимодействие с предметом на локации (предмет должен быть в белой обводке)
- I - открыть инвентарь
- Q - кинуть кидаемый предмет. Если зажать и есть купленная пассивка - будет брошено 3 предмета
- H - лечение
- L - зажечь свет (если есть свеча в инвентаре)
- 1 - активация активного перка, если есть
- Esc - открыть окно меню/закрыть окно

Весь проект является прототипом и не использовался для какой-либо коммерции\
